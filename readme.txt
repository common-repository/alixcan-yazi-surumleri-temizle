== Description ==

WordPress'in Oluşturduğu Otomaik Yazı Sürümlerini Temizlemenizi Sağlar.
Admin Panelde Oluşan Menüye Giderek Yazı Sürümlerini Kapatmadan Kolayca Temizlemenize Yarayan Ufak Bir Eklenti.

== Installation ==

1. Upload `alix-taslak-temizle` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin in the 'Options' (or 'Settings') screen by following the onscreen prompts.

## Documentation
If you need more help installing and configuring the plugin, [see here for more information](http://www.alixcan.net/?p=867).
